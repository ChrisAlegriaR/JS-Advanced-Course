This repository is composed of three parts:

1. **JavaScript Advanced Course**:  
   In the **`course/`** folder, you will find my personal notes on advanced JavaScript concepts that I learned in the **SergieCode** course. This course provides the fundamental knowledge needed to understand and practice **Object-Oriented Programming (OOP)** in JavaScript, including detailed explanations about paradigms, classes, constructors, instances, methods, encapsulation, inheritance, polymorphism, abstraction, static members, and error handling.

2. **JavaScript Advanced Examples**:  
   Instead of separate practice folders, different examples were developed directly inside the course files to reinforce the main advanced JavaScript concepts. These examples were used as training to strengthen understanding of OOP behavior, class relationships, method execution, and internal JavaScript mechanics through hands-on code application.

3. **JavaScript Advanced Concepts**:  
   Throughout the course, multiple advanced JavaScript concepts were explained and demonstrated using structured theory and commented code. These concepts focus on understanding how JavaScript applies object-oriented principles and how these ideas are used in real-world programming scenarios.
